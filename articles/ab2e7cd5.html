centOS7安装mysql

发表于 2021-03-18 更新于 2021-03-21 分类于 大数据
多多复习加练习

Linux虚拟机中安装mysql命令初步总结：

  • 检查本机有没有安装mysql：rpm -qa | grep mysql

    • 如果有，请先卸载：rpm -e “有关mysql的文件”；卸载之后我们find / -name mysql,用来删除所有查询到的和mysql有关的文件；

    • 如果没有，我们要从 官网上下载对应的mysql安装包；通过xshell或其他终端控制工具使用rz命令把那个下载到本地的mysql安装包上传到虚拟机器中，上传的文件夹即执行rz命令的文件夹，然后安装（有的下载的是安装包，有的下载的是压缩包，建议下载安装包）：yum localinstall mysql80-community-release-el7-3.noarch.rpm;

    • 安装完成后，再执行yum install mysql-community-server即完成安装mysql；

  • 启动mysql:service mysqld start;

  • 查询mysql服务是否启动：ps -ef | grep mysql

  • 准备第一次登陆mysql ：

    • 第一次登陆mysql时候，可以使用mysql的临时密码，查询mysql临时密码：

      ​ grep ‘temporary password’ /var/log/mysqld.log,由于种种原因，我们可能找不到mysqld.log文件，但是我们第一次登陆mysql，没有密码怎么登陆呢？

      解决方案就是首次登陆跳过密码验证：

      ①首先我们要打开my.cnf文件 vi /etc/my.cnf, 然后加入skip-grant-tables这句话 ，然后重启mysql，这样首次登陆mysql是不需要输入密码的。

      ②登陆命令：mysql -uroot;登陆成功之后就可以修改那个临时密码了

      ③更新mysql的用户表 update mysql.user set password=password(‘要修改的密码’) where user=’root’;

      ④然后刷新权限并退出 flush privilege；最后我们再进入my.cnf文件，把那个skip那条语句注释掉 ，再次⑤重启mysql服务，登陆时就可以用我们修改的那个密码了。

      正常登陆mysql：mysql -uroot -p直接跟上密码就可以登陆了；

  • 这里可以切换数据库的命令 use 加上数据库名就可以，那么首先切换到mysql；

  • 然后修改一下user权限 update user set host = ‘%’ where user=’root’;这个命令就是把访问该mysql服务的主机变成所有主机，而不仅仅是localhost；
